question: >-
    How do you convert negative exponents into positive exponents using the reciprocal method?
answer: >-
    To convert negative exponents into positive exponents using the reciprocal method, you can follow these steps:\n\n1. **Identify the Negative Exponent**: When you have an expression with a negative exponent, such as $ a^{-n} $, recognize that this indicates the reciprocal of the base raised to the positive exponent.\n\n2. **Apply the Reciprocal Rule**: The rule states that:\n   $$\n   a^{-n} = \frac{1}{a^n}\n   $$\n   This means you take the base $ a $ and raise it to the positive exponent $ n $, then take the reciprocal of that result.\n\n3. **Example**: For instance, if you have $ 5^{-3} $:\n   - First, apply the reciprocal rule:\n   $$\n   5^{-3} = \frac{1}{5^3}\n   $$\n   - Then calculate $ 5^3 $:\n   $$\n   5^3 = 125\n   $$\n   - Finally, take the reciprocal:\n   $$\n   5^{-3} = \frac{1}{125}\n   $$\n\n4. **Multiple Negative Exponents**: If you have an expression with multiple negative exponents, apply the reciprocal rule to each term. For example, in the expression $ \frac{2^{-2}}{3^{-1}} $:\n   - Convert each negative exponent:\n   $$\n   \frac{2^{-2}}{3^{-1}} = \frac{1}{2^2} \cdot \frac{3^1}{1} = \frac{3}{2^2} = \frac{3}{4}\n   $$\n\n5. **Generalization**: This method can be applied to any base and any negative exponent, ensuring that you always convert to a positive exponent by taking the reciprocal.\n\nBy following these steps, you can effectively convert negative exponents into positive exponents using the reciprocal method.
